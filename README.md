# 🏠 House Price Prediction - California Housing

This project involves building a Machine Learning model to predict median house values in California. It explores data cleaning, visualization, and regression techniques.

## 🚀 Model Performance
Through training a **Random Forest Regressor**, I achieved the following results:
- **R² Score:** 0.8167
- **Mean Absolute Error (MAE):** $31,631

## 📊 Key Visualizations
- **Feature Correlation:** Analyzed how features like 'Median Income' directly impact house prices.
- **Top Predictive Features:** Identified that Location (Longitude/Latitude) and Income are the most significant predictors.

## 🛠️ Tools Used
- **Python** (Pandas, NumPy, Scikit-Learn)
- **Visualization:** Matplotlib & Seaborn
- **Environment:** Google Colab
